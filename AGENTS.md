## TradingView MCP

Use TradingView MCP intentionally, not by default.

TradingView MCP is useful when chart context, Pine drafting, replay review, or screenshots are relevant to the task.

TradingView MCP is unnecessary for note cleanup, file organization, documentation editing, or generic writing. If the task does not need chart context, continue without MCP.

When TradingView MCP is relevant and available, use only the minimum chart context needed for the task.
