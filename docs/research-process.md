# Research Process

1. Define the research question in plain language.
2. Decide whether repository notes are enough or whether chart context is required.
3. If TradingView MCP is relevant and available, gather only the chart context needed.
4. Record findings in clear language that a non-developer can review later.
5. Keep screenshots, replay observations, and Pine ideas tied to the specific question being studied.

## Decision rule

Use current chart context only when needed for the research question.

Do not use TradingView MCP for note cleanup, document revisions, file moves, or prompt editing.
