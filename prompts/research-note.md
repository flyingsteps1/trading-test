# Research Note Prompt

Use this prompt to turn one research question into a short note.

## Use

Summarize the question, what was reviewed, and the current conclusion.

If TradingView MCP is relevant and available, include only the chart context needed to support the note.

Use current chart context only when needed. If the task is editorial, organizational, or based on existing notes, continue without MCP.

Keep it readable for a non-developer.

## Return format

Return exactly these headings:
- Question
- Review source
- Key observation
- Conclusion
- Next step
