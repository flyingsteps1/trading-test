# Chart Review Prompt

Use this prompt for chart-focused review.

## Use

Review the setup, key levels, and what would confirm or invalidate the idea.

If TradingView MCP is relevant and available, use chart context, replay review, screenshots, or Pine drafting only where they improve the analysis.

Use current chart context only when needed. Do not assume MCP must be used if the question can be answered from existing notes.

Keep the write-up concise and plain-language.

## Return format

Return exactly these headings:
- Setup
- Key levels
- What supports the idea
- What breaks the idea
- Next action
