# Documentation Edit Prompt

Use this prompt for README, docs, notes, or prompt-template editing.

## Use

Edit the document for clarity, structure, and consistency.

Do not use TradingView MCP unless the document change specifically depends on chart context.

If the task is note cleanup, file organization, documentation editing, or generic writing, complete it without MCP.

## Return format

Return exactly these headings:
- Goal
- Revised text
- What changed
